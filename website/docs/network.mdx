SwiftGraphQL ships with a fully-featured GraphQL client inspired by [`urql`](https://formidable.com/open-source/urql/). The client itself does not depend on the query-builder, but they work together nicely. We are going to showcase how you can use it with the query-builder, but it shouldn't be hard to use it without it.
